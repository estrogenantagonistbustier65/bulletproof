## 🛠️ How to use it

bulletproof is built around a 12-stage workflow. That makes the process easier to follow and helps keep each step in order.

A typical run may include:

1. Research the task
2. Define the goal
3. Break the work into small parts
4. Write the first version
5. Check the logic
6. Test the result
7. Fix problems
8. Review the code
9. Verify the behavior
10. Prepare the final version
11. Clean up the output
12. Move to production

You do not need to manage each part by hand. The system helps AI agents stay on track and avoid jumping ahead.
